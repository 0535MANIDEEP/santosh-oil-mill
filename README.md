
# Santhosh Oil Mill - Interactive Marketing Platform

A premium, interactive web experience built for **Santhosh Oil Mill, Shankarpally**. This site uses physics-based visuals and educational marketing strategies to drive customer engagement and sales for cold-pressed oils.

## 🚀 Key Features

### 1. Interactive Anti-Gravity Zone
*   **Physics Engine:** Built with `Matter.js`.
*   **Playful Exploration:** Product cards and health tips float in zero-gravity Bubbles.
*   **Interactivity:** Users can drag bubbles or click them to reveal detailed information.
*   **Technology:** Synced physics bodies with React DOM elements for a seamless visual experience.

### 2. Marketing & Education Strategy
*   **Nutritional Comparison:** Dedicated section comparing Cold Pressed vs. Refined oils.
*   **Direct-to-Sale CTAs:** Every product bubble opens a modal with a **one-tap WhatsApp ordering** link.
*   **Trust Metrics:** Highlights local sourcing, traditional wood-pressing, and chemical-free production.

### 3. Technical Excellence
*   **Framework:** Next.js (App Router)
*   **Styling:** Clean, modern Vanilla CSS with a bespoke "Evergreen & Gold" color palette.
*   **Responsive:** Intelligent fallback to a static list view on mobile to ensure fast performance.
*   **SEO:** Pre-configured with local keywords for "Shankarpally Oil Mill".

## 🛠️ Local Development

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Start Development Server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) to see the interactive experience.

3.  **Deploying to Production:**
    Recommended: [Deploy on Vercel](https://vercel.com/new) for best performance and easy custom domain setup.

Deployment
This project is optimized for deployment on Vercel.

Push code to GitHub.
Import project into Vercel.
Deploy.
Live URL: [https://santosh-oil-mill-shankarpally.vercel.app/]

## 📁 Project Structure

*   `/src/app/page.tsx` - Main layout and marketing flow.
*   `/src/components/AntiGravityZone.tsx` - Core physics engine and floating visuals.
*   `/src/components/ComparisonSection.tsx` - Educational marketing content.
*   `/src/lib/data.ts` - Central source of truth for products, benefits, and business info.
*   `/src/app/globals.css` - Design system and premium animations.
